Code Sample: KPI Calculation Pipeline


import pandas as pd

# Load ledger export
ledger = pd.read_csv('data/ledger.csv')
fx = pd.read_csv('data/fx_rates.csv')

# Normalize to USD and filter quarter
q = '2025Q2'
q_ledger = (ledger
    .merge(fx, on='currency')
    .assign(amount_usd=lambda d: d.amount * d.usd_rate)
    .query('quarter == @q'))

# KPIs
gmv = q_ledger.loc[q_ledger.type == 'sale', 'amount_usd'].sum()
refunds = q_ledger.loc[q_ledger.type == 'refund', 'amount_usd'].sum()
net_revenue = gmv - refunds
print({'gmv': gmv, 'refunds': refunds, 'net_revenue': net_revenue})

Financial Math

Discounted Cash Flow (DCF):

\[ V_0 = \sum_{t=1}^{T} \frac{FCF_t}{(1 + r)^t} + \frac{TV_T}{(1+r)^T} \]

Capital Asset Pricing Model (CAPM):

\[ E[R_i] = R_f + \beta_i (E[R_m] - R_f) \]
