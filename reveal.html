Sentiment Analysis of Earnings Call

A Python script snippet used to analyze transcript sentiment.


import pandas as pd
from textblob import TextBlob

# Load the earnings call transcript
with open('q3_transcript.txt', 'r') as file:
    transcript = file.read()

# Perform sentiment analysis
sentiment = TextBlob(transcript)
print(f"Polarity: {sentiment.sentiment.polarity:.2f}")
print(f"Subjectivity: {sentiment.sentiment.subjectivity:.2f}")

Key Financial Metrics

We use the Compound Annual Growth Rate (CAGR) to measure long-term growth.

$$ CAGR = \left( \frac{EV}{BV} \right)^{\frac{1}{n}} - 1 $$

Where:
$EV$ = Ending Value
$BV$ = Beginning Value
$n$ = Number of Years
